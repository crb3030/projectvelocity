How this works: When a law enforcement officer issues a speeding citation between two toll points, log it here. The automated enforcement system checks this log before issuing its own ticket — if the same vehicle was already cited by an officer on the same segment within the last 60 minutes, the automated ticket is suppressed to prevent double-ticketing. The deduplication window can be adjusted in Settings.
